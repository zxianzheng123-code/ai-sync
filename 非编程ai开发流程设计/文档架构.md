# 文档架构

## 文档架构设计逻辑

文档按“项目根目录总纲、参考文档明细、项目治理文档”三层组织。  
项目根目录只放总入口与主线文档；详细说明统一收进 `参考文档/`；阶段性收束、验证、发布规则统一收进 `项目治理/`。

```text
项目根目录/
├── README.md
├── 功能需求.md
├── 主业务流程.md
├── 参考文档/
│   ├── mock原件/
│   │   └── 最终mock/
│   ├── 页面及组件说明/
│   │   └── 应用层/
│   │       ├── 页面清单.md
│   │       ├── 页面流转.md
│   │       ├── 通用组件/
│   │       │   └── <组件名>.md
│   │       ├── 专用组件/
│   │       │   └── <组件名>.md
│   │       ├── 布局说明.md
│   │       └── 对象层/
│   │           ├── 通用组件/
│   │           │   └── <组件名>.md
│   │           ├── 专用组件/
│   │           │   └── <组件名>.md
│   │           ├── 布局说明.md
│   │           └── 功能层/
│   │               ├── 通用组件/
│   │               │   └── <组件名>.md
│   │               ├── 专用组件/
│   │               │   └── <组件名>.md
│   │               └── 布局说明.md
│   ├── 数据说明/
│   │   ├── 数据库架构.md
│   │   ├── 数据字典.md
│   │   ├── 状态字典.md
│   │   ├── A类数据说明.md
│   │   ├── B类数据说明.md
│   │   └── ...
│   └── 技术说明/
│       ├── 技术选型.md
│       ├── 项目文件架构.md
│       └── 功能模块架构.md
└── 项目治理/
    ├── 基线修订记录.md
    ├── 测试与回归清单.md
    └── 发布前检查与版本节奏.md
```

## 文档类型说明

| 文档类型         | 文件名             | 章节                                 |
| ------------ | --------------- | ---------------------------------- |
| `项目入口`       | `README.md`     | `项目目标`、`当前范围`、`文档入口`、`核心文档入口`      |
| `功能需求`       | `功能需求.md`       | `功能目标`、`功能范围`、`核心功能`、`不做项`         |
| `主业务流程`      | `主业务流程.md`      | `主流程`、`支线`、`输入`、`处理`、`输出`、`反馈`     |
| `mock原件`     | `最终mock/`       | `最终视觉真源`、`最终交互真源`                  |
| `页面清单`       | `页面清单.md`       | `页面名称`、`页面职责`、`上游输入`、`下游输出`        |
| `页面流转`       | `页面流转.md`       | `入口页面`、`跳转条件`、`关键状态`、`主路径`         |
| `布局说明`       | `布局说明.md`       | `布局口径`                             |
| `通用组件`       | `<组件名>.md`      | `结构说明`、`交互说明`、`动效说明`               |
| `专用组件`       | `<组件名>.md`      | `结构说明`、`交互说明`、`动效说明`               |
| `数据库架构`      | `数据库架构.md`      | `对象`、`存储位置`、`是否派生`、`是否持久化`         |
| `数据字典`       | `数据字典.md`       | `字段归属`、`字段含义`、`来源`、`去向`            |
| `状态字典`       | `状态字典.md`       | `状态对象`、`状态值`、`进入条件`、`退出条件`         |
| `分类数据说明`     | `<分类名>数据说明.md`  | `核心对象`、`源数据`、`派生数据及派生规则`           |
| `技术选型`       | `技术选型.md`       | `前端`、`后端`、`数据库`、`鉴权`、`AI调用`、`部署方案` |
| `项目文件架构`     | `项目文件架构.md`     | `前端目录`、`后端目录`、`本地资源`、`外部依赖`        |
| `功能模块架构`     | `功能模块架构.md`     | `模块逻辑图`、`功能模块一览表`                  |
| `基线修订记录`     | `基线修订记录.md`     | `变更项`、`变更原因`、`影响范围`、`生效时间`         |
| `测试与回归清单`    | `测试与回归清单.md`    | `主流程检查项`、`边界状态检查项`、`设计一致性检查项`      |
| `发布前检查与版本节奏` | `发布前检查与版本节奏.md` | `发布检查`、`版本分级`、`发布节奏`、`可发布标准`       |
