# 前端Mock 矩阵


## 维度

| 维度           | 只负责什么                           |
| ------------ | ------------------------------- |
| `布局mock`     | 页面级布局，从骨架布局推进到页面布局、组件布局         |
| `固定显示组件mock` | 固定显示的组件本身长什么样、需要哪些状态、状态怎么罗列     |
| `触发显示组件mock` | 触发后才显示的组件在什么条件下出现、由什么触发、触发后显示什么 |
| `数据mock`     | 页面和组件里承载什么内容与数据状态               |
| `视觉mock`     | 静态上看起来怎样：颜色、字体、间距、边框、阴影、层级      |
| `交互mock`     | 不同层级对象上的交互：从应用到对象、功能、组件，再到数据运算  |
| `技术mock`     | 这版 mock 在实现层面有什么约束、边界和风险        |

## 维度节点

| 维度           | 文档  | 节点                                                          |
| ------------ | --- | ----------------------------------------------------------- |
| `布局mock`     |     | `应用导航及区域布局 -> 应用对象导航布局 -> 对象功能导航布局 -> 固定显示组件布局 -> 动态显示组件布局` |
| `固定显示组件mock` |     | `添加占位说明 -> 结构 -> 状态罗列 -> 定稿`                                |
| `触发显示组件mock` |     | `添加占位说明 -> 结构 -> 触发条件 -> 定稿`                                |
| `数据mock`     |     | `展示数据 -> 运算数据 -> 测试数据`                                      |
| `视觉mock`     |     | `黑白线稿 -> 基础样式 -> 静态高保真 -> 动态高保真`                            |
| `交互mock`     |     | `应用层 -> 对象层 -> 对象功能层 -> 静态组件层 -> 动态组件层 -> 数据运算层`            |
| `技术mock`     |     | `HTML -> CSS -> 少量原生JS -> 静态设计指导依赖 -> 全局设计指导依赖`             |

## 轮次推进

| 轮次  | 布局mock    | 固定显示组件mock | 触发显示组件mock | 数据mock | 视觉mock | 交互mock | 技术mock   |
| --- | --------- | ---------- | ---------- | ------ | ------ | ------ | -------- |
| 第一轮 | 应用导航及区域布局 | 添加占位说明     | 添加占位说明     | 示例数据   | 黑白线稿   | 应用层    | HTML     |
| 第二轮 | 应用对象导航布局  | 结构         | 结构         | 示例数据   | 基础样式   | 对象层    | CSS      |
| 第三轮 | 对象功能导航布局  | 状态罗列       | 触发条件       | 真实数据   | 静态高保真  | 对象功能层  | 少量原生JS   |
| 第四轮 | 固定显示组件布局  | 定稿         | 触发条件       | 真实数据   | 静态高保真  | 静态组件层  | 静态设计指导依赖 |
| 第五轮 | 动态显示组件布局  | 定稿         | 定稿         | 测试数据   | 动态高保真  | 动态组件层  | 全局设计指导依赖 |
| 第六轮 | 动态显示组件布局  | 定稿         | 定稿         | 测试数据   | 动态高保真  | 数据运算层  | 全局设计指导依赖 |

## 参考文档

### 参考文档设计逻辑

参考文档按“界面消费口径、数据真源口径、技术全局口径”三类分层组织：页面及组件说明承接界面结构与组件，数据说明承接数据对象与派生规则，技术说明承接全局实现结构。

```text
参考文档/
├── mock原件/
│   └── 最终mock/
├── 页面及组件说明/
│   └── 应用层/
│       ├── 通用组件/
│       │   └── <组件名>.md            # 结构说明、交互说明、动效说明
│       ├── 专用组件/
│       │   └── <组件名>.md
│       ├── 布局说明.md
│       └── 对象层/
│           ├── 通用组件/
│           │   └── <组件名>.md
│           ├── 专用组件/
│           │   └── <组件名>.md
│           ├── 布局说明.md
│           └── 功能层/
│               ├── 通用组件/
│               │   └── <组件名>.md
│               ├── 专用组件/
│               │   └── <组件名>.md
│               └── 布局说明.md
├── 数据说明/
│   ├── 数据库架构.md                  # 对象 / 存储位置 / 是否派生 / 是否持久化
│   ├── A类数据说明.md                 # 核心对象、源数据、派生数据及派生规则
│   ├── B类数据说明.md
│   └── ...
└── 技术说明/                         # 复杂项目建议保留
    ├── 项目文件架构.md                # 前端、后端、本地资源、外部依赖的大目录划分
    └── 功能模块架构.md                # 模块逻辑图、功能模块一览表：名称 / 功能 / 归属 / 依赖
```

### 参考文档类型说明

| 文件类型     | 文件名            | 章节                          |
| -------- | -------------- | --------------------------- |
| `mock原件` | `最终mock/` | `最终视觉真源`、`最终交互真源` |
| `布局说明`   | `布局说明.md`      | `布局口径`                      |
| `通用组件`   | `<组件名>.md`     | `结构说明`、`交互说明`、`动效说明`        |
| `专用组件`   | `<组件名>.md`     | `结构说明`、`交互说明`、`动效说明`        |
| `数据库架构`  | `数据库架构.md`     | `对象`、`存储位置`、`是否派生`、`是否持久化`  |
| `分类数据说明` | `<分类名>数据说明.md` | `核心对象`、`源数据`、`派生数据及派生规则`    |
| `项目文件架构` | `项目文件架构.md`    | `前端目录`、`后端目录`、`本地资源`、`外部依赖` |
| `功能模块架构` | `功能模块架构.md`    | `模块逻辑图`、`功能模块一览表`           |
