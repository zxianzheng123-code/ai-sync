# 服装图片商品化 PLAYBOOK

把随手拍的服装平铺照片，自动转化为电商级纯白底商品图。

---

## 1. 目标

| 项 | 说明 |
|----|------|
| 输入 | 文件夹 A 中的随手拍服装平铺照片 |
| 输出 | 文件夹 B 中的电商级纯白底商品图 |
| 执行者 | AI Agent，全自动无人值守 |

## 2. 铁律

以下规则不可违反，优先级高于一切处理标准。

1. **忠实原物**：输出图中的服装必须与原图是同一件——款式、结构、细节不可篡改。
2. **细节保真**：纽扣、logo、面料纹理、缝线等细节必须保留，不可模糊、丢失或虚构。
3. **校色不改色**：允许校正白平衡和色温偏差，使颜色接近自然光下的真实观感；禁止改变服装本身的色相（如蓝变绿、红变橙）。
4. **姿态可调不可改**：允许将皱褶拉平、摆放整理；禁止改变服装的款式轮廓。
5. **模糊必须清晰化**：原图模糊的区域必须增强清晰度，输出图不允许存在模糊区域。

## 3. 输入规范

| 项 | 要求 |
|----|------|
| 来源 | 文件夹 A |
| 格式 | PNG 或 JPG |
| 内容 | 单件服装，平铺拍摄 |
| 背景 | 不限（会被替换） |

### 跳过条件

遇到以下情况，跳过该文件并记录原因：

- 非 PNG/JPG 格式的文件。
- 图片中包含多件服装。
- 图片中无可识别的服装。

## 4. 输出规范

| 项 | 要求 |
|----|------|
| 目标 | 文件夹 B |
| 格式 | PNG |
| 尺寸 | 1200 × 1200 像素 |
| 命名 | 与原文件同名（扩展名统一为 `.png`） |
| 背景 | 纯白（#FFFFFF） |

## 5. 处理标准

每条标准只在此处定义，全文其他位置仅引用编号。

| 编号 | 步骤 | 标准 | 关联铁律 |
|------|------|------|----------|
| P1 | 去除背景 | 将原始背景完全移除，替换为纯白底（#FFFFFF）。服装边缘干净，无残留杂物或抠图痕迹。 | — |
| P2 | 清晰化 | 对模糊区域进行增强，输出图整体清晰锐利。 | 铁律 5 |
| P3 | 校色 | 校正白平衡和色温偏差，还原自然光下的真实颜色。不可改变色相。 | 铁律 3 |
| P4 | 姿态整理 | 将皱褶拉平、摆放摆正，呈现平整的平铺展示效果。不可改变款式轮廓。 | 铁律 4 |
| P5 | 居中构图 | 服装在画面中居中，四周白色边距每侧 ≥60px，各侧边距差值 ≤20px。 | — |
| P6 | 缩放输出 | 调整为 1200×1200 像素。服装不可被裁切。导出为 PNG。 | 铁律 1 |

## 6. 质量验收

### 6.1 样品基准

- 样品图路径：`文件夹 B/_sample/sample.png`。
- 样品图代表最终成品的标杆质量。
- 若样品图不存在，中止全部处理，报告错误。

### 6.2 合格定义

Q1–Q6 全部通过 = 合格。任一不通过 = 不合格。

| 编号 | 验收条件 | 关联标准 |
|------|----------|----------|
| Q1 | 背景纯白，无残留杂物或抠图痕迹 | P1 |
| Q2 | 清晰度不低于样品图 | P2 |
| Q3 | 服装颜色与原图一致，色相未被改变 | P3 |
| Q4 | 服装细节与原图一致，未丢失或虚构 | 铁律 1, 2 |
| Q5 | 构图居中，边距符合 P5 | P5 |
| Q6 | 服装完整，未被裁切 | P6 |

### 6.3 不合格处理

- 不合格图片不输出到文件夹 B。
- 记录文件名及不合格条件编号（Q1–Q6）。

## 7. 工作流程

AI Agent 按以下步骤顺序执行：

### 7.1 初始化

1. 确认样品图 `文件夹 B/_sample/sample.png` 存在。不存在 → 中止，报告错误。
2. 读取文件夹 A，筛选 PNG 和 JPG 文件。
3. 跳过不符合输入规范（第 3 章）的文件，记录跳过原因。
4. 输出待处理文件清单。

### 7.2 逐张处理

对每张图片，按 P1 → P2 → P3 → P4 → P5 → P6 顺序执行处理标准（第 5 章）。

### 7.3 逐张验收

处理完成后，按第 6.2 节逐条验收：

- 合格 → 保存到文件夹 B，文件名与原文件相同（扩展名 `.png`）。
- 不合格 → 按第 6.3 节处理。

### 7.4 生成报告

全部图片处理完毕后，生成处理报告（第 8 章）。

## 8. 处理报告

每次批量处理完成后，输出以下报告：

| 项目 | 内容 |
|------|------|
| 处理总数 | 文件夹 A 中符合输入规范的图片数 |
| 成功 | 数量 + 文件名清单 |
| 失败 | 数量 + 文件名 + 不合格条件编号（Q1–Q6） |
| 跳过 | 数量 + 文件名 + 跳过原因 |
