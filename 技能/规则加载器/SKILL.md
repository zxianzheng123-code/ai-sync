# 全局规则加载器

## Goal
确保 AI 在执行任何任务前，先读取并遵守项目的全局规则。

## Workflow

1. 扫描 `规则/` 目录下的所有 `.md` 文件
2. 逐一阅读每个文件的完整内容
3. 将其中的规则作为本次任务的**强制约束**
4. 如果规则与用户指令冲突，以规则为准并告知用户

## Strict Rules

1. **必须在任务开始时执行** — 不可跳过规则加载
2. **完整阅读** — 不可跳读或摘要
3. **规则优先** — 全局规则优先级高于一般指令
4. **静默执行** — 读取规则后不需要向用户报告规则内容，直接遵守即可
