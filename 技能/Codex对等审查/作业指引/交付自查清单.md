# Codex 对等审查 — 交付自查清单

## 审查完整性
- [ ] 审查对象与用户确认的内容一致
- [ ] Codex 所有审查意见均已逐条处理（采纳或给出保留理由）
- [ ] 讨论未超出上限（讨论 ≤3 轮/次，打回 ≤3 次）

## 最终方案质量
- [ ] 最终方案完整，无遗漏章节或代码段
- [ ] 采纳的修改已正确体现在最终方案中
- [ ] 未引入新的问题或矛盾

## 变更说明
- [ ] 变更说明准确列出了因 Codex 反馈修改的地方
- [ ] 变更说明准确列出了坚持原方案的地方及理由
- [ ] 包含讨论轮次统计

## 透明度
- [ ] 每轮讨论均向用户输出了进度摘要
- [ ] 分歧点均已展示给用户（无黑盒讨论）
- [ ] 超限退出时明确交由用户裁决
