# 技能依赖图

> 本文件定义技能间的数据流向与协作关系。修改任何技能的输入/输出时，须同步更新本图。

## 三层架构总览

```
┌─────────────────────────────────────────────────────┐
│  顶层 T 型核心硬技能                                  │
│  ┌────────────┐ ┌────────────┐ ┌─────────────────┐ │
│  │ 产品决策师  │ │线稿交互原型师│ │ 全栈编码师       │ │
│  │(验证→需求) │ │(蓝图→稿→原型)│ │(选型→蓝图→前后端 │ │
│  │            │ │            │ │ →AI集成)         │ │
│  └────────────┘ └────────────┘ └─────────────────┘ │
├─────────────────────────────────────────────────────┤
│  中层杠杆工具                                        │
│  部署上线师 │ 代码维护师 │ 增长黑客 │ 迭代规划师       │
│  Skill编译器 │ PLAYBOOK编写师 │ Obsidian编辑         │
│  网页正文提取                                        │
├─────────────────────────────────────────────────────┤
│  底层内功引擎                                        │
│  通用任务引擎 │ 通用审查协议                          │
└─────────────────────────────────────────────────────┘
```

## 产品飞轮（数据流拓扑）

```
用户口述想法
     │
     ▼
 产品决策师 ──→ project_context.md
     │                │
     ├────────────────┤
     ▼                ▼
线稿交互原型师    全栈编码师 Phase A
     │                │
     │ layout_blueprint.md    │ infra_spec.md
     │ style_guidebook.pen    │ 项目骨架
     │ static_ui_mock.pen     │
     │ interaction_map.md     │
     │                │
     └───────┬────────┘
             ▼
     全栈编码师 Phase B ──→ code_blueprint.md
             │
     全栈编码师 Phase C ──→ 生产前端代码
             │
     全栈编码师 Phase D ──→ 生产后端代码（仅 full-stack）
             │
     全栈编码师 Phase E ──→ AI 集成（仅含 AI 功能时）
             │
             ▼
         部署上线师 ──→ deploy_config.md + 线上 URL
             │
             ▼
         增长黑客 ──→ growth_plan.md
             │
             ▼
       用户反馈 ──→ 迭代规划师 ──→ project_context.md 增量
             │
             └──→ 回到产品决策师 / 线稿交互原型师 / 全栈编码师（闭环）
```

## 产出物注册表

| 产出物 | 生产技能 | 消费技能 | 路径 |
|--------|---------|---------|------|
| `project_context.md` | 产品决策师 | 线稿交互原型师、全栈编码师、增长黑客 | `design/` |
| `layout_blueprint.md` | 线稿交互原型师 | 全栈编码师 | `design/` |
| `style_guidebook.pen` | 线稿交互原型师 | 全栈编码师 | `design/` |
| `static_ui_mock.pen` | 线稿交互原型师 | 全栈编码师 | `design/` |
| `interaction_map.md` | 线稿交互原型师 | 全栈编码师 | `design/` |
| `infra_spec.md` | 全栈编码师 Phase A | 全栈编码师 Phase B-E、部署上线师、代码维护师 | 项目根目录 |
| `code_blueprint.md` | 全栈编码师 Phase B | 全栈编码师 Phase C-E、部署上线师 | `design/` |
| `deploy_config.md` | 部署上线师 | （终端产物） | 项目根目录 |
| `growth_plan.md` | 增长黑客 | （终端产物，可供迭代规划师参考） | 项目根目录 |

## 独立技能（无产品飞轮数据依赖）

| 技能 | 层级 | 输入 | 输出 |
|------|------|------|------|
| 代码维护师 | 中层 | 任意代码 + 可选 infra_spec | 修复/重构/优化/测试后的代码 |
| PLAYBOOK编写师 | 中层 | 用户口述领域知识 | PLAYBOOK 文件 |
| Skill编译器 | 中层 | PLAYBOOK + 技能需求 | SKILL.md |
| Obsidian编辑 | 中层 | .md / .base / .canvas 文件 | 编辑后的文件 |
| 网页正文提取 | 中层 | URL | Markdown 正文 |
| 通用任务引擎 | 底层 | 任意任务描述 | 动态交付物 |
